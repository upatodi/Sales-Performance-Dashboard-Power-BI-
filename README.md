# Sales-Performance-Dashboard-Power-BI-

Project Overview

This project presents an interactive Sales Performance Dashboard built using Power BI to analyze sales, profit, customer segments, and monthly trends.
The dashboard is designed to help business stakeholders monitor performance, identify profitable areas, and support data-driven decision-making.

Business Objectives

Track overall sales and profit performance
Analyze customer segments and product categories
Identify high- and low-performing product lines
Understand monthly sales trends and seasonality
Enable quick insights through interactive filtering

Tools & Technologies

Power BI
Data Modeling
DAX (basic measures)
Data Visualization & Dashboard Design

Key Metrics (KPIs)

Total Sales
Total Profit
Total Customers
Total Products

Monthly Sales Trend
